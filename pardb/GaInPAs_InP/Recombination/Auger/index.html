[top] Auger Recombination model parameters from sdevice

FILE: Default.tcl
EXPRESSION: 

Auger recombination rate is calculated using
R_Auger = ( C_n n + C_p p ) ( n p - ni_eff^2)
with C_n,p = (A + B (T/T0) + C (T/T0)^2) (1 + H exp(-{n,p}/N0))

CALCULATED PARAMETER: A_n, A_p, B_n, B_p, C_n, C_p, H_n, H_p, N0_n, N0_p
