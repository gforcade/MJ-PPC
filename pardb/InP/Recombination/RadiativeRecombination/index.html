[top] Room temperature Radiative Recombination model parameters from Levinshtein

FILE: Levinshtein.tcl
EXPRESSION: 

Radiative recombination rate is calculated using
R_Radiative = C * (T/Tpar)^alpha * (n p - ni_eff^2)
Temperature dependence is suppressed by setting alpha=0

CALCULATED PARAMETER: C and alpha
VALIDITY: This model is valid only for T=300K
REFERENCE:
"Handbook series on Semiconductor Parameters, Vol I" M. Levenshtein, S. Rumyantsev, M. Shur, World Scientific, 1996
NOTES:
