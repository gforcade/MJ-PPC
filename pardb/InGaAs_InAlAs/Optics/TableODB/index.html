[top] Room temperature model from Bhattacharya

FILE: Bhattacharya.tcl
EXPRESSION: 


Room temperature values of wavelength, refractive index and extinction coefficient are created using Bhattacharya.csv in Bhattacharya folder

CALCULATED PARAMETER: llist, nlist and klist return wavelength [um], refractive index and extinction coefficient respectively.
VALIDITY: this model is valid only for T=300K and In mole fraction of 0.3
REFERENCE:
"Properties of Lattice-Matched and Strained Indium Gallium Arsenide, (EMIS Datareviews Series No. 8)," Bhattacharya, P., INSPEC 1993
