[top] Room temperature model from Levinshtein

FILE: djurisic.tcl
EXPRESSION: 


Room temperature values of wavelength, refractive index and extinction coefficient are created using INSPEC.csv in INSPEC folder

CALCULATED PARAMETER: llist, nlist and klist return wavelength [um], absorption coefficient [cm^-1], refractive index and extinction coefficient respectively.
VALIDITY: this model is valid only for T=300K
REFERENCE:
A.B.Djurisic et.al.J.Appl.Phys Vol.86 Nr.1 1999
NOTES: To use this model, use the keyword "SemAbs(model=ODB)" in the appropriate section of the Sentaurus Device input file
