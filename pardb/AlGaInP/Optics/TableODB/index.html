[top] Room temperature model from Sopra database (obtained from Refractiveindex.info, reference: Kato et al. (1993)

FILE: Sopra.tcl
EXPRESSION: 


Room temperature values of wavelength, refractive index and extinction coefficient are created using on of the data files in Sopra folder for molar fractions 0 (GaInP), 0.1, 0.3, 0.6, 0.7, 1.0 (AlInP)

CALCULATED PARAMETER: llist, nlist and klist return wavelength [um], refractive index and extinction coefficient respectively.
VALIDITY: This model is valid only for (AlxGa1-x)50In50P: 0 (GaInP), 0.1, 0.3, 0.6, 0.7, 1.0 (AlInP)
