[top] BGN model from Blieske

FILE: Blieske.tcl
EXPRESSION: 

Bandgap narrowing  is calculated using
deltaEg = A_i N^(1/3) + B_i N^(1/4) + C_i N^(1/2) + D_i N^(1/2)
where i is n for n-type or p for p-type

CALCULATED PARAMETER: A_i, B_i, C_i, D_i
VALIDITY: This model is valid for both n-GaAs and p-GaAs
REFERENCE:
U. Blieske, Konzentratorsolarzellen aus Galliumarsenid: Modul- und Tandem-Anwendung, Vol. 333, VDI-Verlag GmbH, DÃ¼sseldorf, 1995

[top] Jain Roulston model from sdevice

FILE: Default.tcl
EXPRESSION: 

Bandgap narrowing  is calculated using
deltaEg = A_i N^(1/3) + B_i N^(1/4) + C_i N^(1/2) + D_i N^(1/2)
where i is n for n-type or p for p-type

CALCULATED PARAMETER: A_i, B_i, C_i, D_i
VALIDITY: This model is valid for both n-GaAs and p-GaAs
REFERENCE:
S.C.Jain and D.J.Roulston, "A Simple Expression for Band Gap Narrowing (BGN) in Heavily Doped Si, Ge, GaAs and Ge(x)Si(1-x) Strained Layers," Solid-State Electronics, vol. 34, no.5, pp.453-465, 1991

[top] Band gap Narrowing model from Schubert

FILE: Schubert.tcl
EXPRESSION: 

Bandgap narrowing  is calculated using


deltaEg = A_i*N^(1/3)

where i is n for n-type or p for p-type

CALCULATED PARAMETER: A_i
VALIDITY: This model is valid for both n-GaAs and p-GaAs
REFERENCE:
E.F.Schubert, Physical Foundations of Solid-State Devices, 2006

[top] Doping dependent model

FILE: Tiwari.tcl
EXPRESSION: 

Bandgap narrowing  is calculated using
deltaEg =C_p*N^(1/2)

CALCULATED PARAMETER: C_p
VALIDITY: This model is valid only for p-GaAs

S.C.Jain and D.J.Roulston, "A Simple Expression for Band Gap Narrowing (BGN) in Heavily Doped Si, Ge, GaAs and Ge(x)Si(1-x) Strained Layers," Solid-State Electronics, vol. 34, no.5, pp.453-465, 1991
