[top] High field mobility model with default Transferred Electron model parameters from sdevice

FILE: Default-TEM2.tcl
EXPRESSION: 

High field mobility is calculated using
mu_highfield = (mu_lowfield+(vsat/E)*(E/E0_TrEf)^4)/(1+(E/E0_TrEf)^4)
The saturation velocity, vsat is calculated using
vsat = A_vsat - B_vsat*(T/T0)

CALCULATED PARAMETER: Transferred electron model parameters and model 2 parameters for velocity saturation model
VALIDITY: This model is valid at high electric fields
REFERENCE:
Sentaurus Device User Guide
NOTES: To use this model, activate Transferred Electron model in sdevice command file
