[top] Temperature dependent model from Palankovski

FILE: Palankovski.tcl
EXPRESSION: 

Temperature dependent mobility is calculated using
mu_const = mumax (T/T0)^(-Exponent)

CALCULATED PARAMETER: mu_const_n and mu_const_p returns temperature dependent mobility for n and p type carriers respectively.
VALIDITY:
REFERENCE:
"Analysis and Simulation of Heterostructure Devices" V. Palankovski, R. Quay, Springer-Verlag/Wien, 2004 http://www.iue.tuwien.ac.at/phd/palankovski/
NOTES: This model is temperature dependent and doping concentration independent. The mobility value is the low doping mobility value from the doping dependent Sotoodeh model

[top] Temperature dependent model from sdevice

FILE: Madelung.tcl
EXPRESSION: 

Temperature dependent mobility is calculated using
mu_const = mumax (T/T0)^(-Exponent)

CALCULATED PARAMETER: mu_constant_n and mu_constant_p returns mobility for n and p type carriers respectively.
VALIDITY: The electron mobility model is valid for temperatures in the range 77-300 K The hole mobility model is valid for temperatures in the range 100-300 K
REFERENCE:
"Semiconductors:Data Handbook", O. Madelung, Springer, 2004
NOTES: This is a temperature dependent and doping concentration independent model.
