[top] Temperature dependent model from sdevice

FILE: Default.tcl
EXPRESSION: 

Temperature dependent mobility is calculated using
mu_const = mumax (T/T0)^(-Exponent)

CALCULATED PARAMETER: mu_constant_n and mu_constant_p returns constant mobility for n and p type carriers respectively.
VALIDITY:
REFERENCE:
Gergoe's Material Parameter Database
NOTES: This is a temperature dependent and doping concentration independent model.
