[top] Temperature and Doping dependent model from Levinshtein

FILE: Default.tcl
EXPRESSION: 

Temperature and doping dependent minority carrier lifetime is calculated using,
tau = taumin + ( taumax - taumin ) / ( 1 + ( N/Nref )^gamma)
where,
tau(T) = tau * ( (T/300)^Talpha )
or
tau(T) = tau * exp( Tcoeff * ((T/300)-1) )

CALCULATED PARAMETER: taumin_i, taumax_i, Nref_i, gamma_i, Talpha_i, Tcoeff_i, Etrap, where i is n for n-type or p for p-type
